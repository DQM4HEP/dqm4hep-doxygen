Detailed Description

WsServer class.

Implements a websocket server with which users can:

  • create services to broadcast data to all clients
  • receive commands from clients
  • receive requests from clients

template<typename Operation >
void dqm4hep::net::WsServer::synchronize ( Operation  operation)
inline

Synchronize a user operation with the server thread.

This can be used for updating shared data between the user code and the request and command handlers. Typical use is by using a lambda function.

Example:

server->synchronize([&](){
shared_data = update();
});

The performed operation has to be fast in order to avoid long server operation blocking.

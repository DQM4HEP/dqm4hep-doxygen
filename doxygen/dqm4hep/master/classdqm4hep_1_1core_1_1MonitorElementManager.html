Create a ROOT histogram and wrap it into a monitor element.

The created histogram is owned by the monitor element and must not be delete by users. The allocation is performed using the ROOT object constructor possibly with arguments. For scalar types (int, double, ...) use the bookScalar() method instead. For other custom classes, use the bookObject() function. The last argument args are the arguments passed to the histogram constructor after the "name" and "title" arguments.

Example:

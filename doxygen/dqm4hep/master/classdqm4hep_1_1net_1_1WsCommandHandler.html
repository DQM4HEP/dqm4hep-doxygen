Detailed Description

WsCommandHandler class.

Implement a command handler pattern where a client sends data to the server that receives it. No response is sent back to the client. To receive the data from the client you can use the signa provided by the onCommand() method:

auto handler = server->createCommandHandler();
handler->onCommand().connect(&object, &Object::method);

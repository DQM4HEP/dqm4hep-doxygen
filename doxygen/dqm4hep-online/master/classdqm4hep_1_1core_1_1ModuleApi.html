Detailed Description

ModuleApi class.

Responsible for direct operation on module application and modules such as monitor element booking and directory navigation. Monitor element booking should be done via a Module.

Example :

MonitorElement *pMyRealHistogram1DElement = NULL;
ModuleApi::bookRealHistogram1D(this, pMyRealHistogram1DElement, "MyHisto", "My histo title", 10, 1, 10);
